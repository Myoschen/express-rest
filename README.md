# express-rest

> Build a Restful api from scratch with Express, PostgreSQL

## Usage

```shell
# Clone all project
git clone https://github.com/willy14620/express-rest.git

# Install dependencies
npm ci
```

## Resource

| Method     | URI                      |
|------------|--------------------------|
| **GET**    | /api/article             |
| **POST**   | /api/article             |
| **PUT**    | /api/article/:article_id |
| **DELETE** | /api/article/:article_id |
